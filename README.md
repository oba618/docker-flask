# #格言共有(Docker-Flask)

座右の銘とそれにまつわるエピソードの共有が出来ます。

## 概要

#格言共有のフロント側のレポジトリです。Docker、Flask、Vueを使用して作成しています。

[バックエンド側はこちら](https://github.com/oba618/share-adage-service)

## 構築環境

- Docker: 20.10.7
- python: 3.8.11
- flask: 2.0.1
- vue: 3.2.31

## 設置ページ

[#格言共有](https://share-adage-service.herokuapp.com/)
